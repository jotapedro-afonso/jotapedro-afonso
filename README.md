# 📌 Projetos

## 🏥 Sa.Med — Sistema de gerenciamento para profissionais da saúde

Projeto desenvolvido como **Trabalho de Conclusão de Curso (TCC)**.

O Sa.Med tem como objetivo auxiliar profissionais autônomos da área da saúde na organização da rotina profissional, oferecendo recursos como:

- gerenciamento de agenda;
- controle de horários;
- organização de clientes;
- melhoria na gestão de tempo.

Tecnologias utilizadas:

- HTML
- CSS
- JavaScript
- Node.js
- SQL Express
- Figma
- Whimsical
- Visual Studio Code
